部署说明：开发与验收环境采用单机部署模式。后端 Spring Boot 服务（:8080）、前端 Vite 开发服务器（:5173）和 KingbaseES 金仓数据库（:54321）均运行在同一台 Windows 或 Linux 电脑上。WS63 设备通过 WiFi 接入互联网并连接华为云 IoTDA，IoTDA 通过 AMQP 队列将设备数据推送至后端。浏览器可在本机（localhost）或局域网其他设备通过 IP 访问前端页面。AI 功能需要后端能够访问 Dify API 地址（HTTP 请求）。前端通过 Vite proxy 将 /api 请求代理到后端 :8080 端口，后端通过 JDBC 连接 KingbaseES :54321，同时使用本地 SQLite 文件（sensor_data.db）作为业务数据源。
