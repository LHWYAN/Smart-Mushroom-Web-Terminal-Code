表关系说明：device_info 为核心实体，通过 device_id 字段与 sensor_data（1:N）和 device_commands（1:N）建立外键关联。删除设备时由 DeviceService.delete() 主动级联删除关联的传感器数据和命令记录。users 表独立于设备业务关系，专用于用户认证，通过 JdbcTemplate 原生 SQL 访问。四张表均建在 KingbaseES 金仓数据库上，满足"金仓数据库上建 3 个以上数据表"的任务要求（实际 4 张）。
建表方式：前三张表由 schema-kingbasees.sql 脚本初始化（DROP + CREATE + INSERT 默认设备）；users 表由 AuthService.initTable() 在系统启动时通过 CREATE TABLE IF NOT EXISTS 自动建表。
