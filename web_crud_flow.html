Web CRUD 操作流程图
登录鉴权前置 → 前端发起 → 后端处理 → 数据库操作 → 响应返回

前端
用户访问系统
路由守卫 beforeEach
检查 localStorage Token
有有效 Token?
无 Token / 过期
跳转 /login
登录页面
POST /auth/login
用户名 + 密码
KingbaseES 验证
BCrypt + JWT 签发
获取 Token
存入 localStorage
有有效 Token
Axios 拦截器注入

进入业务页面

↓ ↓ ↓ 登录成功后，携带 Token 发起 CRUD 操作 ↓ ↓ ↓
